# Galaxy-Classifier-Using-ML

This a galaxy classifier ML model that uses a CNN to classify galaxies based on their shapes. It is being trained on 24,000 images from SDSS. 
It will be used to classify uncertain galaxies, which are taken from 240,000 galaxy clasfications by Galaxy Zoo.
